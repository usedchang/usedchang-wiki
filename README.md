# usedchang-wiki

XCPC 题解博客，记录算法学习与刷题历程。

## 在线访问

**网站**: https://usedchang.netlify.app/

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **部署平台**: Netlify

## 本地开发

```bash
# 进入项目目录
cd vue-app

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
vue-app/
├── src/
│   ├── views/      # 页面组件
│   ├── router/     # 路由配置
│   ├── utils/     # 工具函数
│   └── assets/    # 静态资源
└── public/         # 公共资源
```

## 功能特性

- 主页展示最新题解
- 学习计划页面
- Codeforces 统计
- 题解编辑与发布
- 多主题切换（Academic / Modern / 深色 / 护眼）
